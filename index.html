2. Variables and Data Types

            

                
In Python, variables are used to store data. Let's learn about different types of variables:


                
Example:

print("Name:", name)
print("Age:", age)

3. Basic Operators

            

                
Python allows you to perform basic mathematical operations. Let's learn how:


                
Example:

                

# Addition
a = 5 + 3
print(a)

# Subtraction
b = 10 - 2
print(b)

# Multiplication
c = 4 * 3
print(c)

4. Control Flow (If Statements)

            

                
Python uses if statements to make decisions in your code based on conditions.


                
Example:

                

age = 18
if age >= 18:
    print("You're an adult.")
else:
    print("You're a minor.")
